We are pleased to announce the latest updates and enhancements to the BootcampsHub App. This release focuses on improving functionality, fixing bugs, and enhancing the user experience.

## **BootcampsHub App Development:**

---

### New Fixes and Enhancements:
1. **Video Title and Chapter Name Overlap**  
   - Resolved an issue where video titles were overlapping the chapter names within the program module.

2. **App Stability Improvements**  
   - Fixed multiple crashing issues within the chat functionality, improving overall app stability.

3. **Audio Recording Issue**  
   - Addressed the problem preventing audio recordings from functioning properly in the chat feature.

4. **Day2Day Activities Card Layout**  
   - Fixed padding issues on both the left and right sides of the Day2Day Activities card, ensuring proper alignment.

5. **GitHub Icon Alignment**  
   - Corrected the GitHub icon to match the design consistency with Facebook and Instagram icons.

6. **Recording Time Display**  
   - Fixed an issue where the recording time counter was not updating properly when a recording was started.

7. **Push Notifications**  
   - Ongoing work on improving push notification functionality to keep users informed and engaged.

---

Thank you for using the Bootcampshub App! We hope these updates improve your experience. If you have any feedback or encounter any issues, please do not hesitate to contact our support team.
